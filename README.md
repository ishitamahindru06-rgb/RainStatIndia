# RainStatIndia
RainStat India — Rainfall Trend &amp; Variability Analyzer
A fully interactive, web application for analyzing India's rainfall patterns, seasonal trends, and historical variability across all 28 states.

CHE110 Course Project | Built with HTML, CSS & JavaScript

📌 Features

🗺️ Interactive Map — Leaflet.js powered map of India with state-level rainfall indicators

🔍 State Search Engine — Autocomplete search across all 28 Indian states with quick-select chips
📊 Multi-Chart Analysis per state:
Monthly rainfall distribution
Historical trend 
Seasonal anomaly detection
Seasonal breakdown (Pre-Monsoon, Monsoon, Post-Monsoon, Winter)

📋 District-wise Data Table — Sortable rainfall data by district
🌐 National Overview — Country-level long-period average (LPA) comparison chart
🌙 Dark / Light Theme Toggle
🌧 Animated Rain Canvas — Decorative background animation
📱 Fully Responsive — Works on desktop, tablet, and mobile with hamburger nav
⏳ Scroll Progress Bar — Visual reading progress indicator

📊 Data Source
Rainfall values are based on open repositories , Long Period Averages (1951–2000). State and district-level figures are representative estimates derived from IMD baseline data and are used for educational and visualization purposes.

🔭 Future Scope

🌐 Live IMD API Integration — Connect to real-time IMD or OpenWeatherMap API for live rainfall data instead of static estimates
🤖 ML-based Rainfall Prediction — Integrate a machine learning model (LSTM / Random Forest) to forecast rainfall for upcoming seasons based on historical trends
🌏 Multi-language Support — Add Hindi and regional language support for wider accessibility across India
📊 Comparison Mode — Side-by-side comparison of two states' rainfall patterns simultaneously
